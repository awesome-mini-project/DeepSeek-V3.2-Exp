## 1. 项目概述

DeepSeek-V3.2-Exp 是一个实验性模型，引入了 DSA (DeepSeek Sparse Attention) 稀疏注意力机制。

核心特点：
- DSA (DeepSeek Sparse Attention): Top-K 稀疏注意力
- MLA (Multi-Head Latent Attention): KV 压缩机制
- MoE (Mixture of Experts): 256 路由专家
- FP8 量化: 块级量化，block_size=128

## 4. kernel.py 详解

### 4.1 act_quant (激活量化)

```
输入: X (M, N) BF16
       │
       ▼
┌────────────────────────────────────────┐
│         act_quant_kernel               │
│  ┌──────────────────────────────────┐  │
│  │ 1. 加载 X 到 Shared Memory       │  │
│  │    x_shared (32, 128)            │  │
│  │                                  │  │
│  │ 2. 计算每行 absmax              │  │
│  │    amax_local (32,)             │  │
│  │                                  │  │
│  │ 3. 计算 scale                   │  │
│  │    s_local (32,)                │  │
│  │    scale = amax / 448           │  │
│  │                                  │  │
│  │ 4. 量化: clamp(X/s, -448, 448)  │  │
│  │    y_local (32, 128) FP8        │  │
│  │                                  │  │
│  │ 5. 写回 Y 和 S                 │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
       │
       ▼
输出: Y (M, N) FP8, S (M, ⌈N/128⌉) FP32
```

### 4.2 fp8_gemm (FP8 矩阵乘法)

```
输入: A (M,K) FP8, B (N,K) FP8
      scales_a (M,⌈K/128⌉), scales_b (⌈N/128⌉,⌈K/128⌉)
       │
       ▼
┌────────────────────────────────────────┐
│         fp8_gemm_kernel               │
│  K 维度循环 (Pipeline 4 stages)        │
│  ┌──────────────────────────────────┐  │
│  │ 1. 加载 A block (32, 128)        │  │
│  │    加载 B block (128, 128)       │  │
│  │                                  │  │
│  │ 2. 加载并合并 scales            │  │
│  │    Scale_C = scales_a × scales_b │  │
│  │                                  │  │
│  │ 3. GEMM: C_local = A @ B^T       │  │
│  │    (32, 128) FP8                 │  │
│  │                                  │  │
│  │ 4. 累加: C_accum += C × Scale   │  │
│  │    (32, 128) FP32                │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
       │
       ▼
输出: C (M, N) BF16/FP32
```

### 4.3 fp8_index (DSA Indexer)

```
输入: q (B,M,H,D) FP8, q_s (B,M,H) FP32
      k (B,N,D) FP8, k_s (B,N) FP32
       │
       ▼
┌────────────────────────────────────────┐
│         fp8_index_kernel             │
│  ┌──────────────────────────────────┐  │
│  │ 1. 加载 Query (一次全部)         │  │
│  │    q_smem (64, 128)             │  │
│  │                                  │  │
│  │ 2. 循环 Key (每128个)            │  │
│  │    ┌──────────────────────────┐  │  │
│  │    │ 加载 k block (128,128) │  │  │
│  │    │                         │  │  │
│  │    │ GEMM: logits = k @ q^T │  │  │
│  │    │   (128, 64)            │  │  │
│  │    │                         │  │  │
│  │    │ ReLU × q_s             │  │  │
│  │    │                         │  │  │
│  │    │ Reduce sum (head 维度) │  │  │
│  │    │                         │  │  │
│  │    │ × k_s (E8M0 量化)      │  │  │
│  │    └──────────────────────────┘  │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
       │
       ▼
输出: o (B,M,N) FP32 (Index Score)
```

## 5. model.py 核心模块

### 5.1 Indexer (DSA Indexer)

```
输入: x (B,S,d), qr (B,S,q_rank)
      start_pos, freqs_cis, mask
       │
       ▼
┌────────────────────────────────────────┐
│              Indexer.forward()          │
│  ┌──────────────────────────────────┐  │
│  │ Q 路径:                          │  │
│  │   wq_b → reshape → RoPE (非交错)  │  │
│  │        → Hadamard → FP8 量化      │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │ K 路径:                          │  │
│  │   wk → LayerNorm → RoPE (非交错) │  │
│  │        → Hadamard → FP8 量化      │  │
│  │        → 写入 k_cache            │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │ Index Score 计算:                │  │
│  │   weights_proj → 归一化          │  │
│  │   fp8_index(q, weights, k, k_s)  │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │ Top-K 选择:                      │  │
│  │   topk(index_score, k=2048)     │  │
│  │   → topk_indices (B,S,K)         │  │
│  │                                 │  │
│  │   Broadcast 验证多卡一致性        │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
       │
       ▼
输出: topk_indices (B,S,K)
```

### 5.2 MLA (Multi-Head Latent Attention)

```
输入: x (B,S,d), start_pos, freqs_cis, mask
       │
       ▼
┌────────────────────────────────────────┐
│              MLA.forward()             │
│                                          │
│  ┌──────────────────────────────────┐  │
│  │ Q 路径:                           │  │
│  │   wq_a → q_norm → wq_b           │  │
│  │   → split(q_nope, q_pe)          │  │
│  │   → RoPE q_pe                    │  │
│  │   → cat(q_nope, q_pe)            │  │
│  └──────────────────────────────────┘  │
│                                          │
│  ┌──────────────────────────────────┐  │
│  │ KV 路径 (压缩):                   │  │
│  │   wkv_a → split(kv, k_pe)        │  │
│  │   kv → kv_norm                   │  │
│  │   k_pe → RoPE                    │  │
│  │   kv → FP8 量化                  │  │
│  │   → 写入 kv_cache                │  │
│  └──────────────────────────────────┘  │
│                                          │
│  │                                          │
│  ▼  (分支: mask 存在 → Prefill, 否则 → Decode)  │
│                                          │
│  ┌──────────────────────────────────┐  │
│  │ Prefill 分支 (MHA Dense):        │  │
│  │   wkv_b 解压 KV                   │  │
│  │   → 拼接 K                       │  │
│  │   → einsum(Q,K^T) → scores       │  │
│  │   → Indexer Top-K mask          │  │
│  │   → softmax → @V                 │  │
│  └──────────────────────────────────┘  │
│                                          │
│  ┌──────────────────────────────────┐  │
│  │ Decode 分支 (MQA Sparse):        │  │
│  │   wkv_b 预计算反量化              │  │
│  │   → einsum(q_nope, wkv_b)        │  │
│  │     → q_proj (B,S,H,512)         │  │
│  │   → q_proj @ kv_cache.T          │  │
│  │   → q_pe @ pe_cache.T            │  │
│  │   → scores 合并                  │  │
│  │   → Indexer Top-K mask          │  │
│  │   → softmax → @ kv_cache         │  │
│  │   → @ wkv_b_v 投影输出            │  │
│  └──────────────────────────────────┘  │
│                                          │
└────────────────────────────────────────┘
       │
       ▼
输出: h (B,S,d)
```

### 5.3 MoE (Mixture of Experts)

```
输入: x (B,S,d)
       │
       ▼
┌────────────────────────────────────────┐
│                MoE.forward()            │
│  ┌──────────────────────────────────┐  │
│  │ Gate (门控路由):                 │  │
│  │   x @ W^T → scores (B,n_exp)    │  │
│  │   → softmax → topk(k=6)        │  │
│  │   → indices (B,6)               │  │
│  │   → weights (B,6)               │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │ Expert 计算:                      │  │
│  │   对每个本地专家:                │  │
│  │   ┌──────────────────────────┐  │  │
│  │   │ 找到路由到该专家的样本   │  │  │
│  │   │                         │  │  │
│  │   │ w1(x) → SiLU             │  │  │
│  │   │ w3(x)                   │  │  │
│  │   │ SiLU(w1x) × w3x          │  │  │
│  │   │ w2(...)                  │  │  │
│  │   │                         │  │  │
│  │   │ × weights 累加到输出     │  │  │
│  │   └──────────────────────────┘  │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │ Shared Experts (共享专家):       │  │
│  │   处理所有输入                   │  │
│  └──────────────────────────────────┘  │
│                                          │
│  y = sum(expert_outputs) + shared      │
│  └──────────────────────────────────┘  │
│                                          │
│  AllReduce (多卡求和)                   │
└────────────────────────────────────────┘
       │
       ▼
输出: y (B,S,d)
```

### 5.4 Transformer Block

```
输入: x (B,S,d), residual=None
      start_pos, freqs_cis, mask
       │
       ▼
┌────────────────────────────────────────┐
│             Block.forward()             │
│  ┌──────────────────────────────────┐  │
│  │ attn_norm(x, residual)          │  │
│  │   │                              │  │
│  │   ├─ residual=None:              │  │
│  │   │  └→ x_norm, residual=x     │  │
│  │   │                              │  │
│  │   └─ residual!=None:            │  │
│  │      └→ x_norm, residual 保持不变 │  │
│  └──────────────────────────────────┘  │
│       │                                │
│       ▼                                │
│  ┌──────────────────────────────────┐  │
│  │ attn(x_norm, ...)               │  │
│  │   │                              │  │
│  │   └─> MLA (含 Indexer)          │  │
│  └──────────────────────────────────┘  │
│       │                                │
│       ▼                                │
│  ┌──────────────────────────────────┐  │
│  │ ffn_norm(x, residual)           │  │
│  │   │                              │  │
│  │   └─> 合并: x + residual        │  │
│  │       归一化                      │  │
│  │   │                              │  │
│  │   └─> x_norm, residual          │  │
│  │       (residual 仍是原始值)      │  │
│  └──────────────────────────────────┘  │
│       │                                │
│       ▼                                │
│  ┌──────────────────────────────────┐  │
│  │ ffn(x_norm)                     │  │
│  │   │                              │  │
│  │   └─> Layer 0: MLP              │  │
│  │       Layer 1-60: MoE           │  │
│  └──────────────────────────────────┘  │
│                                          │
│  return x, residual                      │
└────────────────────────────────────────┘
```

## 符号说明

- `B`: Batch Size (批大小)
- `S`: Sequence Length (序列长度)
- `d`: Hidden Dimension (隐藏维度, 2048)
- `H`: Number of Heads (头数, 16)
- `D`: Head Dimension (头维度)
- `K`: Key Dimension (KV维度)
- `V`: Value Dimension (Value维度)
- `n_exp`: Number of Experts (专家数, 64)
- `k`: Top-K (选中数量, 2048)
- `end_pos`: 当前序列末尾位置

## 数据类型说明

- `BF16`: BFloat16 (2 bytes)
- `FP8`: Float8 E4M3 (1 byte)
- `FP32`: Float32 (4 bytes)
